    Welcome

    A Bootcamp is one of the most intense experiences you will ever endure. The Coder Academy Flex Track program is no different. Our aim is to get you from ‘zero’ to coding ‘hero’ in the shortest amount of time. So buckle up - get ready to be immersed & overwhelmed :) … it is going to be a journey of a lifetime …

    Bootcamp Overview

    The three buttons here provide the key information you need to be successful in the subject.

    • The Subject Outline button provides a succinct overview of the subjects covered, learning outcomes and assessable items.
    • The Syllabus button takes you to a list of the topics.
    • The Assignments button takes you to the place where all your assessment information and submission points can be found.

    Navigation

    To the left, you'll see a list of links. These links will take you to specific sections of the course. Some of the important links are:

    • Home: This is where you are now. Select the Home button whenever you need to access the Subject Outline or quick links to each module.
    • Assignments: The same as the button above. This is is where you will find assignment/assessment information, which includes briefs, submission points and instructions. In some cases, it's also where you will go to log into live exams.
    • Marks: This is where you'll find a list of your grades and marks when assignment results have been finalised.
    • Modules: This link will take you to the list of weekly modules for the subject. Links to these are also at the bottom of this page. 
    • Find a Resource: This is a link to a library page, where you can search for learning materials.
    • Conferences: This is where you can log into live lectures or watch videos of the session that have been recorded.

    Below, are links to each module of this subject. To begin, simply select the module you're up to.

    Jump In

    The bootcamp consists of 4 terms with 10 weeks each. Each week, you will find the lecture videos, learning resources and learning activities required for the subjects. Each week will be progressively released during the course. Once the course officially commences, you will be able to click on the current week to get started.
